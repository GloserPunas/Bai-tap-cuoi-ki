-----------------------------------------CatRunner-------------------------------------------------------

DESCRIPTION:

YOu will play as a cat which must dodge obstacles on his home way.

HOW TO PLAY:

- Use KEY UP, KEY DOWN to move the cat.
- Don't let the cat get hit by the obstacle!

REQUIRED:
 - Code::Block
 - SDL Libarary
 - SDL_ttf
 - SDL_image


-------------------------------------------------Enjoy!------------------------------------------------------
